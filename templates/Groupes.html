Utilisation du Code Structure :

Nous devons escalader en délégation pour modifier l`état d`une session d`inscription en terminée.
Le routage doit se faire vers la délégation qui a mis en place et organisé la formation en question.
Pour ce faire nous devons nous baser sur le code structure de la formation pour identifier la délégation organisatrice de la formation.

Exemple formation : 84:SXNAA221 => 84 = code structure
